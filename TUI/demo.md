# MediaCrawler TUI 演示说明

## 安装

```bash
# 安装 Textual 框架
uv pip install textual

# 或使用 pip
pip install textual
```

## 运行

```bash
# 从项目根目录运行
python TUI/run.py

# 或从 TUI 目录运行
cd TUI
python app.py
```

## 使用流程

1. **启动程序**
   ```bash
   python TUI/run.py
   ```

2. **选择平台**
   - 点击"小红书"按钮

3. **配置 Creator IDs**
   - 在输入框输入 Creator ID
   - 按 Enter 添加
   - 点击列表项删除

4. **运行爬虫**
   - 点击"运行爬虫"按钮
   - 爬虫在后台启动

5. **返回或退出**
   - 点击"返回"回到主界面
   - 按 `Q` 键退出程序

## 支持的 Creator ID 格式

- 完整 URL: `https://www.xiaohongshu.com/user/profile/6129d00a0000000001002b57`
- 纯 ID: `6129d00a0000000001002b57`
- 用户 ID: `7797463367` (微博)

## 配置文件修改

程序会自动修改对应的配置文件：

```python
# config/xhs_config.py
XHS_CREATOR_ID_LIST = [
    "https://www.xiaohongshu.com/user/profile/6129d00a0000000001002b57",
    "6129d00a0000000001002b58",
]
```

## 运行的命令

```bash
uv run main.py --platform xhs --lt qrcode --type creator --save_data_option postgres
```

可根据需要修改 `TUI/app.py` 中的命令参数。
