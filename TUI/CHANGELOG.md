# 更新日志

## [1.0.0] - 2025-01-28

### 新增
- ✨ 初始版本发布
- ✨ 支持 7 个平台的配置界面（小红书、抖音、快手、B站、微博、贴吧、知乎）
- ✨ 可视化 Creator ID 列表管理（添加/删除）
- ✨ 配置即时保存到对应的配置文件
- ✨ 一键启动爬虫功能
- ✨ 优雅的 TUI 界面设计

### 技术实现
- 使用 Textual 框架构建
- 支持模态对话框
- 使用 DataTable 组件展示列表
- 正则表达式解析和修改配置文件
- 后台进程管理

### 已知限制
- 爬虫输出不在 TUI 中显示
- 每次只能配置一个平台
- 配置修改无撤销功能
